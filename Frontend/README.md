# Slide Designer Frontend

Next.js frontend with all 12 P0 core features integrated with the TypeScript backend.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the application.

## All P0 Features Integrated

1. Grid Layout System
2. Typography Hierarchy  
3. Color Palettes
4. Chart Components
5. Text Overflow Handling
6. Master Slides
7. Basic Transitions
8. Accessibility Features
9. Export Formats (PDF/PPTX/HTML)
10. Image Optimization
11. Content Validation
12. LLM Judge

## Project Structure

- `app/` - Next.js App Router pages
- `components/features/p0/` - All 12 P0 feature components
- `hooks/` - Custom React hooks
- `__tests__/` - Component tests

## Testing

```bash
npm test
```

## Build

```bash
npm run build
npm start
```
